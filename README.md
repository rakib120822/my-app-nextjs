Foodie is a modern food ordering web application built with Next.js and Tailwind CSS. It focuses on a clean UI, smooth user experience, and beginner-friendly project structure. The app uses mock authentication and Toastify for user feedback


## Features

-  Browse food items with a clean and responsive UI

-  Styled using Tailwind CSS

- Toast notifications using React Toastify

- Mock authentication (login/logout flow)

- Fast performance with Next.js

- Fully responsive (mobile, tablet, desktop)


## Tech Stack


**Server:** Node, Express

**Framework:** Next.js

**Styling:** Tailwind CSS

**Alerts & Notifications:** React Toastify

**Authentication:** Mock Authentication

**Package Manager:** npm** Next.js
